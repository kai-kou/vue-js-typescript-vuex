# vue-js-typescript-vuex

Vue.js+TypeScriptでVuex利用方法の検証。

## Usage

```sh
> git clone https://github.com/kai-kou/vue-js-typescript-vuex.git
> cd vue-js-typescript-vuex/app
> yarn
> yarn serve
```

use Docker  

```sh
> git clone https://github.com/kai-kou/vue-js-typescript-vuex.git
> cd vue-js-typescript-vuex
> docker-compose up -d
> docker-compose exec app bash

# cd app
# yarn
# yarn serve
```
